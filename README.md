# DCD Liver Transplantation Analysis Dashboard

Interactive dashboard analyzing trends in DCD (Donation after Circulatory Death) liver transplantation across 138 US transplant centers from 2019-2025.

## Key Findings

- **+347% DCD growth** from 2019 to 2025 (annualized)
- **32% total volume increase** nationally
- **48% of centers** are "expanders" (growing both DCD and total volume)
- **24% of centers** are "substituters" (replacing DBD with DCD)
- **NRP adoption (56%)** does not predict expansion vs substitution

## Features

- National quarterly trends visualization
- Center-level scatter plot (NRP rate vs DBD change)
- Filterable center table by region, volume tier, and trajectory
- MOBH (Barnes Jewish) highlighted for reference
- Print/PDF optimized styling

## Data Source

SRTR STAR files, 2019 Q1 through 2025 Q3. All 2025 data is annualized from 74.8% complete year.

## Usage

Simply open `index.html` in a web browser. No server required.

To print or save as PDF, click the "Print / PDF" button in the header.
